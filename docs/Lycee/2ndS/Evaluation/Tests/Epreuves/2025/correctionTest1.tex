\question
Compléter les identités suivantes :

\[
(a+t)^{3} = a^{3}+3a^{2}t+3at^{2}+t^{3}, \hspace{2cm} (a-t)^{3} = a^{3}-3a^{2}t+3at^{2}-t^{3}
\]
\[
a^{3}+t^{3} = (a+t)(a^{2}-at+t^{2}), \hspace{2cm} a^{3}-t^{3} = (a-t)(a^{2}+at+t^{2})
\]

%==================== QUESTION 2 ====================
\question
Factoriser les expressions suivantes :

\[
a^2xy + aby^2 + b^2xy + abx^2 = (ax+by)(ay+bx)
\]

\[
3a^2 + 3b^2 - 12c^2 - 6ab = 3\bigl((a-b)^2 - (2c)^2\bigr) = 3(a-b-2c)(a-b+2c)
\]

\[
y^2 - x^2 + 2x - 1 = y^2 - (x^2 - 2x + 1) = y^2 - (x-1)^2 = (y - (x-1))(y + (x-1)) = (y-x+1)(y+x-1)
\]
